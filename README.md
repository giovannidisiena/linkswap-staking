# Contract Summaries

## StakingRewardsFactory

- Creates new staking contracts
- Deploying address is authorised to deploy rewards
- Anyone can start and fund rewards
- Deploying address can emergency withdraw

## StakingRewards

- Handles staking 
- Handles rewards
